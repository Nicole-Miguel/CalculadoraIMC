# Calculadora IMC

Aplicação dotnet console para cálculo do IMC

## Funcionalidades

- Calcula o IMC
- Exibe a faixa de peso
- Calcula o  valor que deve ser ganho ou perdido para chegar no peso ideal

## Tecnologia

- dotnet core 2.1

## Como Executar

Fazer o clone do repositório e executar o comando:

```
dotnet run
```
